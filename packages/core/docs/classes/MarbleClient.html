    Class MarbleClient

    Main SDK entrypoint for interacting with the Marble API.

    Example:

    const marble = new MarbleClient({
      baseUrl: "https://api.marble.io",
      apiKey: "sk_123"
    });
    const posts = await marble.listPosts({ limit: 10 });
