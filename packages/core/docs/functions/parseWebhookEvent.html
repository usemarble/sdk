    • parseWebhookEvent<T>(
          jsonBody: string,

      ): WebhookEvent<T>

      Parse a webhook JSON body into a typed WebhookEvent.

      Type Parameters

      • T

        The desired type of the data payload after mapping.

      Parameters

      • jsonBody: string

        Raw JSON string of the webhook body.

      • mapData: (u: unknown) => T

        A mapper that converts/validates the unknown data into your type T.

      Returns WebhookEvent<T>

      Example

      type PostCreated = { id: string };
      const evt = parseWebhookEvent<PostCreated>(rawJson, (u) => {
        if (!u || typeof (u as any).id !== "string") throw new Error("bad data");
        return u as PostCreated;
      });
      console.log(evt.type, evt.data.id);
